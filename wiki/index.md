## Methods

- **RSSM (Recurrent State Space Model)** — Hybrid deterministic+stochastic latent dynamics backbone; introduced in [[hafner-2019-icml]](papers/hafner-2019-icml.md)
- **DreamerV1/V2/V3** — Actor-critic trained in latent imagination; [[hafner-2019-icml]](papers/hafner-2019-icml.md), [[hafner-2021-iclr]](papers/hafner-2021-iclr.md), [[hafner-2023-arxiv]](papers/hafner-2023-arxiv.md)
- **IRIS** — Discrete autoencoder + autoregressive Transformer world model for Atari 100k; [[micheli-2023-iclr]](papers/micheli-2023-iclr.md)
- **DIAMOND** — Diffusion-based (EDM) world model for pixel-space RL; [[alonso-2024-neurips]](papers/alonso-2024-neurips.md)
- **TD-MPC / TD-MPC2** — Task-oriented latent dynamics with MPPI planning and TD-learning; [[hansen-2022-icml]](papers/hansen-2022-icml.md), [[hansen-2024-iclr]](papers/hansen-2024-iclr.md)
- **TWM (Transformer-based World Model)** — Transformer-XL dynamics replacing RSSM in DreamerV2; [[robine-2023-iclr]](papers/robine-2023-iclr.md)
- **JEPA / H-JEPA** — Joint Embedding Predictive Architecture predicting in representation space; [[lecun-2022-openreview]](papers/lecun-2022-openreview.md)
- **GAIA-1** — 9.6B-parameter multimodal autoregressive world model for autonomous driving; [[hu-2023-arxiv]](papers/hu-2023-arxiv.md)
- **NWM (Navigation World Models)** — Conditional Diffusion Transformer for goal-conditioned visual navigation planning; [[bar-2024-cvpr]](papers/bar-2024-cvpr.md)
- **Cosmos WFM** — Open-weight World Foundation Model platform (diffusion + autoregressive families); [[agarwal-2025-arxiv]](papers/agarwal-2025-arxiv.md)
- **GenRL** — Multimodal-foundation world model connecting VLM latent space to generative world model for embodied RL; [[mazzaglia-2024-neurips]](papers/mazzaglia-2024-neurips.md)
- **LeJEPA / SIGReg** — Provably collapse-free JEPA via Sketched Isotropic Gaussian Regularization (Epps-Pulley CF test); [[balestriero-2025-iclr]](papers/balestriero-2025-iclr.md)
- **LeWorldModel (LeWM)** — End-to-end JEPA world model from raw pixels using SIGReg + action-conditioned Transformer predictor + CEM latent planning; [[maes-2026-arxiv]](papers/maes-2026-arxiv.md)
