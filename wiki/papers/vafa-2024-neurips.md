# Evaluating the World Model Implicit in a Generative Model

> **Vafa et al.** — a Myhill-Nerode-inspired evaluation framework reveals that transformers trained to near-perfect next-token accuracy on NYC taxi shortest paths fail to represent a valid world model (sequence compression precision 0.10, sequence distinction precision 0.35), and that GPT-4 achieves task accuracy 1.00 on logic puzzles but compression precision only 0.21, presented at NeurIPS 2024 (Spotlight).

**Authors:** Keyon Vafa (Harvard), Justin Y. Chen (MIT), Ashesh Rambachan (MIT), Jon Kleinberg (Cornell), Sendhil Mullainathan (MIT) | **Venue:** NeurIPS 2024 (Spotlight) | **arXiv:** [2406.03689](https://arxiv.org/abs/2406.03689)

---

## Problem & Motivation

Generative models trained with next-token prediction objectives can achieve near-perfect accuracy on token-level benchmarks while failing to represent meaningful world models. Standard evaluation metrics — next-token accuracy, held-out perplexity, task-specific accuracy — are insensitive to whether the model has internalized a structured causal model of the world or has instead learned a statistical shortcut. A model that memorizes surface-level n-gram co-occurrence patterns can achieve high next-token accuracy on routing tasks without encoding any notion of map topology. The core issue is that next-token prediction is a local criterion: it verifies that the model assigns probability to correct next steps, but does not verify that the model's internal state is sufficient to determine all future valid continuations from any given point. This paper asks: given a generative model trained on sequences generated by some underlying world, what formal criteria must hold for the model to have implicitly learned that world?

---

## Core Idea

The paper introduces a framework grounded in the **Myhill-Nerode theorem** from formal language theory. For any sequence-generating process with a state space (formalized as a Deterministic Finite Automaton, DFA), two prefixes are **equivalent** if and only if they lead to the same state — which means they have identical sets of valid future continuations. A generative model represents a valid world model if and only if it assigns identical distributions over continuations to prefixes that reach the same world state, and distinct distributions to prefixes that reach different world states. This yields two diagnostic metrics that can be computed empirically without access to the model's internals: **sequence compression** (same-state prefixes → same continuations) and **sequence distinction** (different-state prefixes → different continuations). High next-token accuracy is neither necessary nor sufficient for either metric to be satisfied.

---

## How It Works

### World Model Formalization

The paper formalizes a ground-truth world as a **Deterministic Finite Automaton** (DFA) M = (Q, Σ, δ, q₀, F), where Q is the set of states, Σ is the alphabet (tokens), δ: Q × Σ → Q is the transition function, q₀ is the start state, and F ⊆ Q is the set of accepting states. Given a sequence prefix s = (s₁, ..., s_t), the current state is q_s = δ*(q₀, s) (the state reached by following transitions for each token). The ground-truth continuation distribution C*(s) is defined as the distribution over all valid completions from state q_s.

**Key insight:** Two prefixes s and s' are **Myhill-Nerode equivalent** (s ~ s') iff δ*(q₀, s) = δ*(q₀, s') — i.e., they reach the same DFA state. This equivalence partitions all prefixes by their underlying world state. A generative model P_θ implicitly learns the world model M iff: s ~ s' implies C_θ(s) = C_θ(s') (compression) and s ≁ s' implies C_θ(s) ≠ C_θ(s') (distinction).

### Why Next-Token Accuracy Fails

The paper proves that high next-token accuracy is **insufficient** for world model learning:

- A model that assigns uniform probability over all completions from any prefix achieves next-token accuracy equal to the fraction of valid next tokens at each step, which can be close to 1 in low-branching tasks, but completely fails compression and distinction — it has no state representation at all.
- Conversely, a model could achieve compression but fail distinction (same continuation distribution regardless of state → it has collapsed all states into one).
- The paper also shows that even probing the model's hidden states for the current world state (current-state probe accuracy) can be high while compression and distinction fail — because probes only measure if information is decodable from the model's representations, not whether that information is used consistently to generate valid continuations.

### Sequence Compression Precision

**Definition:** For a set of prefix pairs {(s, s')} where s ~ s' (both reach the same DFA state), sequence compression precision measures the fraction of such pairs for which the model generates the same distribution over continuations. Operationally: sample many length-k completions from P_θ conditioned on s, and from P_θ conditioned on s'. If the completion sets are identical (or have high overlap), compression is satisfied. The metric is precision (not recall) because it measures whether same-state pairs yield the same continuations, not whether different-state pairs yield different continuations.

**Intuition:** If a transformer trained on NYC shortest paths has truly learned the city map, then any two routes that have arrived at the same intersection (same DFA state) should produce the same set of valid onward routes — regardless of how the routes got to that intersection.

### Sequence Distinction Precision/Recall

**Definition:** For prefix pairs {(s, s')} where s ≁ s' (they reach different DFA states), sequence distinction measures whether the model generates different distributions over continuations.

- **Distinction precision:** Among pairs where the model generates different continuations, what fraction are actually at different world states?
- **Distinction recall:** Among pairs that are actually at different world states, what fraction does the model distinguish (generate different continuations for)?

**Intuition:** If two routes end at different intersections in Manhattan, a correct world model should be able to plan different onward routes from each. A model that treats all intersections identically fails distinction completely.

### Graph Reconstruction Algorithm

The paper introduces an algorithm to **reconstruct the implicit map** from a generative model's sequence-level behavior, without accessing the model's parameters. Given samples of valid continuations from many prefixes:

1. For each pair of prefixes (s, s'), compute the overlap of their continuation sets (sampled completions).
2. Build an empirical equivalence graph: s and s' are in the same equivalence class if their continuations overlap above a threshold.
3. Reconstruct the graph topology from the inferred equivalence classes and the observed transitions between classes.

This produces a graph that can be compared visually and topologically to the ground-truth world (e.g., the Manhattan street map). **Impossible structures** in the reconstructed map — edges crossing streets at wrong angles, streets with wrong cardinal orientations, flyover connections between non-adjacent nodes — directly indicate where the model's implicit world model violates physical topology.

## Experiments

### NYC Taxi: Shortest Paths

**Setup:** The ground-truth world is the **Manhattan street network** — a graph with 4,580 nodes and 9,846 edges extracted from 2014 NYC taxi trajectory data (3,358,737 training sequences, ~120M tokens). Each sequence is a shortest path between two intersections on the Manhattan grid. The generative model is a **GPT-2 Small** (89.3M parameters: 12 layers, 768 hidden dim, 12 heads) trained on these sequences. Trained on 8 A100 GPUs.

**Results (Table 1):**

| Metric | Generative Model | True World Model |
|--------|-----------------|-----------------|
| Next-token test accuracy | 1.00 | 1.00 |
| Current-state probe accuracy | 0.91 | 1.00 |
| Sequence compression precision | **0.10** | 1.00 |
| Sequence distinction precision | 0.35 | 1.00 |
| Sequence distinction recall | 0.20 | 1.00 |

**Interpretation:** The transformer achieves near-perfect next-token accuracy (1.00) and reasonable current-state probe accuracy (0.91), yet sequence compression precision is only 0.10 — meaning in 90% of cases where two prefixes arrive at the same intersection, the model generates different (inconsistent) continuations from them. Distinction recall of 0.20 means that for 80% of pairs that are actually at different intersections, the model incorrectly treats them identically.

**Map reconstruction:** The reconstructed implicit map has physically impossible structures relative to Manhattan: streets with wrong cardinal orientations (an east-west street appearing as north-south), flyover connections (a node appearing connected to a geographically distant node with no street between them), and missing connections. These artifacts confirm that the model's internal representation is not a topographically faithful world model.

### NYC Taxi: Detour Fragility (Table 2)

**Setup:** To further probe the world model, the paper applies **random detours** — forced deviations from the optimal shortest path by a fraction p of steps. A model with a true world model should recover from detours (any valid path from the detour location is valid onward). A model that has memorized shortest paths will fail.

| Detour fraction p | True WM (valid %) | Shortest paths model (valid %) | Random walks model (valid %) |
|---|---|---|---|
| 0% | 100 | 100 | 100 |
| 10% | 100 | **8** | 100 |
| 25% | 100 | 0 | 100 |
| 50% | 100 | 0 | 100 |
| 75% | 100 | 0 | **100** |

The shortest paths model collapses at even 10% detour rate (only 8% valid), while a model trained on random walks (which can start from any intersection and take any valid sequence of steps) achieves 100% valid completions even at 75% detour rate. This demonstrates that the shortest paths model has learned a task-specific distribution rather than a world model — it cannot generalize to any out-of-distribution starting state.

### Othello: Synthetic vs. Championship Data (Table 3)

**Setup:** The Othello board game has a well-defined world (board positions are the DFA states, valid moves are the transition alphabet). Two transformer models are evaluated:

1. **Championship model:** Trained on sequences from recorded human championship games (highly correlated, non-uniform distribution).
2. **Synthetic model:** Trained on uniformly random valid Othello game sequences (no distribution bias).

| Model | Compression Precision |
|---|---|
| Championship | **0.00** |
| Synthetic | **0.98** |

The championship model achieves 0.00 sequence compression precision — it completely fails to represent the Othello world model, even though it likely achieves high next-move accuracy on championship-style play. The synthetic model trained on uniformly random sequences achieves 0.98 — near-perfect world model compression. This demonstrates that **training data distribution** critically determines whether a generative model learns a world model, independent of model capacity or accuracy on the training task.

### LLM Logic Puzzles: GPT-4 and Others (Figure 4)

**Setup:** A set of formal logic puzzles are used where the ground-truth world is fully specified by logical rules (the DFA is the state space of the logic system). Four LLMs are evaluated: GPT-4, GPT-3.5-Turbo, Claude-2, and Llama-2-70B-Chat.

**Results:**

| Model | Task Accuracy | Compression Precision | Distinction Recall |
|---|---|---|---|
| GPT-4 | **1.00** | **0.21** | 0.56 |
| GPT-3.5-Turbo | ~0.90 | ~0.18 | ~0.50 |
| Claude-2 | ~0.85 | ~0.15 | ~0.48 |
| Llama-2-70B | ~0.70 | ~0.10 | ~0.40 |

GPT-4 achieves perfect task accuracy (1.00) but sequence compression precision of only 0.21 and distinction recall of 0.56. **No LLM exceeds compression precision 0.40 or distinction recall 0.60.** This confirms that even state-of-the-art LLMs with high task accuracy do not reliably learn valid world models of formal logical systems.

## Comparison to Prior Work

| Method | What It Measures | Sensitivity to World Model Structure |
|--------|-----------------|--------------------------------------|
| Next-token accuracy | Local step correctness | None |
| Held-out perplexity | Average sequence likelihood | None |
| Task-specific accuracy | End-to-end task performance | Low |
| Linear probing (current state) | Representation decodability | Medium (necessary but not sufficient) |
| Othello board probe (Li et al.) | Internal board state | Medium (probes representation, not behavior) |
| **Sequence compression (ours)** | Same-state → same continuations | Direct |
| **Sequence distinction (ours)** | Diff-state → diff continuations | Direct |

**Li et al. (Othello-GPT probing):** The prior work showed that a linear probe can recover board positions from a transformer trained on Othello move sequences. The present paper shows that probing (whether internal representations encode world states) is insufficient: the championship Othello model likely has decodable board representations, yet achieves compression precision 0.00. The model represents the board state but does not use it consistently to generate valid continuations.

**Burns et al. / Nanda et al.:** Prior work showing emergent world representations in transformers focuses on internal probes. This paper argues that behavioral evidence (continuation distributions) is the correct criterion — probes are necessary but not sufficient.

**vs [[ha-2018-neurips]] (World Models):** Ha & Schmidhuber's world model learns an internal representation for policy training in imagination. Vafa et al.'s framework provides a formal way to evaluate whether such a model has truly internalized environment dynamics versus merely memorizing trajectory patterns.

**vs [[hafner-2023-arxiv]] (DreamerV3):** DreamerV3 evaluates world models via downstream task reward. This paper argues reward-based evaluation is structurally insensitive to world model quality — a model can achieve high reward by exploiting shortcuts without learning the true dynamics.

## Theoretical Contribution

The paper's formal contribution is a constructive proof that the Myhill-Nerode theorem provides the correct necessary and sufficient conditions for a generative model to be a valid world model:

**Theorem (informal):** A generative model P_θ is equivalent to world model M if and only if for all prefix pairs (s, s'), s ~ s' (Myhill-Nerode equivalent) ⟺ C_θ(s) = C_θ(s') (identical continuation distributions).

The proof constructs an explicit correspondence between the model's induced prefix equivalence classes and the DFA's state space. This means the number of distinct states the model represents is exactly the number of distinct continuation distribution classes it exhibits — providing a principled way to count "how many world states" a model has internalized, independently of its parameter count or architecture.

**Myhill-Nerode boundary and interior:** The paper distinguishes:
- **Interior sequences:** prefixes that are strictly in the interior of the DFA (every valid continuation can itself be extended to a valid accepting sequence). Interior sequences have large, structured continuation sets.
- **Boundary sequences:** prefixes near the accepting/terminal states where the continuation set is small. Boundary sequences are easier to memorize individually, explaining why models can achieve high accuracy near the end of sequences (when few tokens remain) while failing world model criteria for interior states.

## Strengths
- Provides the first theoretically grounded formal criterion — derived from the Myhill-Nerode theorem — for evaluating world model learning in generative models, rather than relying on indirect proxies.
- Metrics are model-agnostic and do not require access to model internals — they only require sampling from the model's output distribution, making them applicable to black-box LLMs.
- The graph reconstruction algorithm provides an interpretable, visual artifact (the implicit map) that makes world model failures intuitive to inspect.
- Detour fragility experiment provides a practical test with clear real-world implications: a model that fails at 10% detours is brittle for any deployment involving distribution shift.
- Othello championship vs. synthetic result provides a clean demonstration that training data distribution — not model capacity — determines world model learning.
- NeurIPS 2024 Spotlight recognition validates the significance of the contribution.

## Weaknesses & Limitations
- The framework requires knowing the ground-truth DFA (or a faithful approximation), which is only available in formal domains (navigation, board games, logic puzzles). Extension to unstructured language modeling is not demonstrated.
- Computing the metrics requires sampling many completions per prefix, which is computationally expensive for large LLMs at scale (though the paper notes this can be parallelized).
- The distinction between "representation" (probing) and "use" (behavioral consistency) is conceptually clear but the paper does not fully characterize under what training conditions a model transitions from using its representations consistently.
- The framework treats the world as a DFA with a finite state space — worlds with continuous or infinite state spaces (e.g., physical dynamics) require approximations.
- The NYC taxi experiment is limited to shortest-path routing; whether the findings extend to other graph-structured tasks (e.g., road network planning with multiple valid routes) is not fully explored.

## Key Takeaways
- Perfect next-token accuracy is not evidence of world model learning: a transformer achieving 1.00 next-token test accuracy on NYC shortest paths has sequence compression precision of only 0.10.
- GPT-4 achieves 1.00 task accuracy on formal logic puzzles but only 0.21 compression precision — indicating no LLM tested (GPT-4, GPT-3.5-Turbo, Claude-2, Llama-2-70B) reliably learns a valid world model of logical systems.
- Training data distribution determines world model learning more than model capacity: a synthetic Othello model achieves 0.98 compression precision; a championship-data Othello model achieves 0.00.
- Detour fragility exposes the brittleness of models that memorize surface distributions: the shortest-paths transformer collapses at 10% detours (8% valid), while a random-walks model maintains 100% validity at 75% detours.
- The Myhill-Nerode theorem provides the correct formal criterion: same-state prefixes → same continuations (compression) and different-state prefixes → different continuations (distinction). High accuracy on either one alone is insufficient.
- Graph reconstruction from continuation distributions reveals physically impossible implicit maps (flyovers, wrong orientations) — interpretable evidence that models learn topologically incorrect world representations.
